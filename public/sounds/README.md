# Sound Effects for Supercooked

## Required Sound Files

Place your sound effect files in this directory:

### Footsteps (Required)

- `footstep1.mp3` - Left foot step
- `footstep2.mp3` - Right foot step (or same as footstep1)

### UI Sounds (Optional)

- `click.mp3` - Button clicks
- `select.mp3` - Character selection
- `whoosh.mp3` - Menu transitions

### Gameplay Sounds (Optional)

- `pickup.mp3` - Picking up items
- `place.mp3` - Placing items
- `chop.mp3` - Chopping ingredients
- `cook.mp3` - Cooking sounds

## Recommended Sources

1. **Kenney.nl** (Best match for blocky aesthetic)

   - https://kenney.nl/assets/interface-sounds
   - https://kenney.nl/assets/impact-sounds
   - License: CC0 (Public Domain)

2. **Freesound.org**

   - https://freesound.org/
   - Search: "footstep", "walking", "click"
   - License: Check individual sounds (usually CC0 or CC-BY)

3. **Pixabay**

   - https://pixabay.com/sound-effects/
   - License: Free for commercial use

4. **Mixkit**
   - https://mixkit.co/free-sound-effects/
   - License: Free for commercial use

## File Format

- **Preferred**: MP3 or OGG (best browser support)
- **Size**: Keep files small (<50KB per sound)
- **Quality**: 128kbps is sufficient for game sound effects

## Example Download Instructions

### For Kenney Sounds:

1. Go to https://kenney.nl/assets/interface-sounds
2. Click "Download" (it's free!)
3. Extract the ZIP file
4. Copy desired sounds to this `public/sounds/` folder
5. Rename them to match the names above

### Quick Start:

If you don't have sounds yet, the game will work without them - they're optional!
